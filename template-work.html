PROJECTS

PROJECT 1

This project is an interactive toy aimed at helping children learn basic mathematics, by inserting numbers into its respective slots. The toy uses force sensors to detect when a number has been inserted into the slot, a proximity sensor to turn on/off the toy when someone is within 40cm, and touch sensors to allow users to select the level of difficulty.
